# 🏋️‍♂️ TreinoFlow - Evolução de Carga

O **TreinoFlow** é um web app minimalista focado no registro e acompanhamento de progressão de carga e intensidade de treinos. Com um design "White Gelo" e detalhes em azul, ele oferece uma interface limpa e funcional para atletas que buscam monitorar seu desempenho.

![Status do Projeto](https://img.shields.io/badge/Status-Em_Desenvolvimento-blue)
![Licença](https://img.shields.io/badge/License-MIT-green)

## ✨ Funcionalidades

- **Registro de Exercícios:** Interface intuitiva para salvar carga (kg) e data.
- **Gráficos Automáticos:** Visualização em tempo real da evolução da força através do Chart.js.
- **Persistência Local:** Os dados são salvos no `localStorage` do navegador, não sendo necessário banco de dados externo para uso pessoal.
- **Design Minimalista:** Estética "Gelo & Azul" focada em reduzir distrações.

## 🎨 Interface

O projeto utiliza uma paleta de cores moderna:
- **Fundo:** `#F4F7F6` (White Gelo)
- **Destaques:** `#2563EB` (Azul Vibrante)
- **Cards:** `#FFFFFF` (Branco Puro)

## 🚀 Tecnologias Utilizadas

* [Tailwind CSS](https://tailwindcss.com/) - Framework CSS utilitário para estilização rápida.
* [Chart.js](https://www.chartjs.org/) - Biblioteca JavaScript para criação de gráficos flexíveis.
* [JavaScript ES6+](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) - Lógica de manipulação de DOM e armazenamento local.
